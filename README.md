# Pharo AI Assistant

AI-assisted code completion and documentation generator for Pharo.

## Project Overview
This project aims to build an AI assistant inside Pharo to help developers:
- Generate code suggestions
- Auto create documentation
- Help beginners understand classes
- Improve development speed

## Features
- AI Code Completion
- Documentation Generator
- UI Panel inside Pharo
- Beginner-friendly hints

## Technologies
- Python
- AI / NLP
- Pharo
- Smalltalk
- REST API

## Timeline
- Research and planning
- AI engine prototype
- Documentation generator
- UI integration
- Testing and improvements
